# 歡迎來到 MkDocs

如需完整文檔，請訪問 [mkdocs.org](https://www.mkdocs.org)。

## 命令

* `mkdocs new [dir-name]` - 創建一個新項目。
* `mkdocs serve` - 啟動實時重新加載文檔伺服器。
* `mkdocs build` - 構建文檔站點。
* `mkdocs -h` - 打印幫助信息並退出。

## 項目佈局

```
mkdocs.yml      # 配置文件。
docs/
    index.md    # 文檔主頁。
    ...         # 其他 Markdown 頁面、圖像和其他文件。
```
